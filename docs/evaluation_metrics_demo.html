Trang này lấy đúng output từ cell evaluate trong notebook: so sánh Popularity, Content, CF (SVD), Hybrid trên tập val_relevant (relevant & unseen).

Flow xử lý của cell evaluate

B1: Tạo val_positive với rating >= 4 trên tập validation.
B2: Với mỗi user, bỏ các item đã thấy trong train -> tạo val_relevant.
B3: Với mỗi model, lấy top-K recommendation, tính P@K / R@K / NDCG@K, gom Coverage và thời gian.
B4: Đánh giá thêm riêng nhóm cold-start user (user not in train).
users có relevant unseen: 14129 cold-start users trong val: 3787

Metric định nghĩa

Precision@K = |TopK ∩ Relevant| / K
Recall@K = |TopK ∩ Relevant| / |Relevant|
NDCG@K = DCG / IDCG (có trọng số vị trí)
Coverage = số item được recommend / tổng số item catalog
