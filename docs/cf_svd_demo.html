Demo trực quan: Collaborative Filtering + TruncatedSVD

Mục tiêu: minh họa luồng trong code của bạn: ma trận user-item → center theo user mean → factor hóa latent (SVD) → score = user_vec · item_vec + user_mean → top gợi ý.

Điều khiển demo

Reproducible khi giữ cùng seed Demo latent đơn giản để dễ hiểu

1) Ma trận rating gốc (R)

Ô trống = chưa có tương tác. Đây tương đương dữ liệu từ train_interactions.

2) Ma trận sau khi center theo user mean

Giống đoạn code centered_vals = rating - user_mean.

3) Latent vector user

Vai trò tương tự user_factors[u_idx].

4) Latent vector item

Vai trò tương tự các cột trong item_factors.

5) Công thức dự đoán

score(u, i) = user_vec(u) · item_vec(i) + user_mean(u)

6) Xếp hạng top-N món chưa xem

Trong code thật: bỏ món đã seen, lấy top theo điểm cao nhất.

Gợi ý thuyết trình: "TruncatedSVD giữ lại một số chiều ẩn quan trọng (n_components), nên mô hình gọn hơn nhưng vẫn nắm được pattern sở thích chính."
