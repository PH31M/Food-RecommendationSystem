Cell 4 — Collaborative Filtering (SVD) Plan

Matrix Factorization dùng TruncatedSVD
Ý tưởng

Phân tích ma trận rating R (sparse, 27K×72K) thành U×Σ×VT. Dự đoán rating bằng cách nhân user vector với recipe vector.

Giới hạn

Chỉ hoạt động cho users có trong train set (27,021 users). Cold-start users cần fallback.

# Plan implementation # 1. Build user-item sparse matrix user_map = {uid: i for i,uid in enumerate(train["user_id"].unique())} item_map = id2idx # from NB03 R = sparse.csr_matrix( (ratings, (user_indices, item_indices)), shape=(n_users, n_items) ) # 2. Center by user mean user_means = R.sum(axis=1) / R.getnnz(axis=1) R_centered = R - user_means # 3. Fit SVD svd = TruncatedSVD(n_components=50) U_reduced = svd.fit_transform(R_centered) VT = svd.components_ # (50 × n_items) # 4. Predict pred = U_reduced[user_idx] @ VT + user_mean # → sort, exclude seen, return top-N

Tại sao TruncatedSVD?

Sparse-friendly

Hoạt động trực tiếp trên scipy.sparse, không cần convert dense (tiết kiệm RAM).

Nhanh

Dùng randomized algorithm, nhanh hơn full SVD rất nhiều cho ma trận lớn.

scikit-learn native

Không cần cài thêm library. Đã có sẵn trong requirements.

def recommend_hybrid(user_id, n=10, constraints=None): # 1. Xác định user tier if user_id not in train_user_ids: return recommend_popular(n) # cold-start user = user_features[user_features["user_id"]==user_id] tier = user["activity_level"].values[0] # 2. Lấy scores từ từng model content_scores = recommend_for_user_content(user_id, n=100) cf_scores = recommend_cf(user_id, n=100) # 3. Normalize scores về [0,1] content_norm = min_max_scale(content_scores) cf_norm = min_max_scale(cf_scores) # 4. Weighted combination alpha = 0.7 if tier == "warm" else 0.5 hybrid = alpha * content_norm + (1-alpha) * cf_norm # 5. Apply constraints + return top-N if constraints: hybrid = apply_constraints(hybrid, **constraints) return hybrid.nlargest(n)

8. Bảng thuật ngữ

Thuật ngữGiải thích
TF-IDFTerm Frequency–Inverse Document Frequency. Biến text thành số: từ quan trọng (nhiều trong 1 doc, ít toàn corpus) → trọng số cao.
Cosine SimilarityĐo góc giữa 2 vectors. Giá trị 0→1. So sánh 2 recipes dựa trên TF-IDF vectors.
Sparse MatrixMa trận phần lớn = 0. Dùng scipy.sparse chỉ lưu ô khác 0 → tiết kiệm RAM.
Matrix FactorizationR ≈ U×VT. Mỗi user/recipe = k latent factors. Dự đoán rating bằng dot product.
TruncatedSVDThuật toán phân tích ma trận sparse. Nhanh hơn full SVD, native scikit-learn.
Cold StartUser/recipe mới không có interaction → CF không hoạt động. Giải pháp: Popularity hoặc Content-Based.
Data LeakageThông tin từ test/val set rò rỉ vào training → model "gian lận". Giải pháp: split trước khi tính features.
Bayesian SmoothingRecipe 1 rating = 5.0 không nên rank cao hơn recipe 100 ratings trung bình 4.8.
Hybrid ModelKết hợp Content + CF + Popularity theo context (cold/warm/active) để bù trừ điểm yếu.
Latent FactorsĐặc trưng ẩn model tự học. Factor 1 ≈ "ngọt", factor 2 ≈ "nhanh", v.v.
Precision@KTrong K gợi ý, bao nhiêu % user thực sự thích?
Recall@KTrong tất cả items user thích, tìm được bao nhiêu %?
NDCG@KItems đúng có ở vị trí cao không? (xét thứ tự ranking)
CoverageBao nhiêu % tổng items xuất hiện trong recommendation? Đo đa dạng.
n-gramUnigram = 1 từ ("sugar"), Bigram = 2 từ ("brown sugar"). Dự án dùng (1,2).
Sublinear TF1+log(tf) thay tf thô. Giảm bias từ lặp quá nhiều lần.
